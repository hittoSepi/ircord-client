# Commands

## Chat
- `/msg <user> <text>` тАФ Send a direct message
- `/me <action>` тАФ Send an action message

## Channels
- `/join <#channel>` тАФ Join a channel
- `/part [#channel]` тАФ Leave current or specified channel
- `/names` тАФ List online users

## Voice
- `/call <user>` тАФ Start a voice call
- `/accept` тАФ Accept incoming call
- `/hangup` тАФ End current call
- `/voice` тАФ Toggle voice chat
- `/mute` тАФ Toggle microphone mute
- `/deafen` тАФ Toggle audio deafen
- `/ptt` тАФ Toggle push-to-talk mode

## Security
- `/trust <user>` тАФ Trust a user's identity key

## Utility
- `/search <query>` тАФ Search message history
- `/clear` тАФ Clear message view
- `/settings` тАФ Open settings screen
- `/version` тАФ Show client and server version
- `/status` тАФ Show connection status
- `/help [topic]` тАФ Show help
- `/reload_help` тАФ Reload help files from disk

## Connection
- `/disconnect` тАФ Disconnect from server
- `/quit` тАФ Exit IRCord
